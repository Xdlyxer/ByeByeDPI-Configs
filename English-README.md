## 📖 Detailed Instructions

### 1. Download ByeByeDPI
- Official Repository: [ByeByeDPI on GitHub](https://github.com/romanvht/ByeByeDPI)
- Download the latest APK and install it on your phone.

**Important!**
Choose the version carefully to avoid issues:
- For **TVs**, choose the second version from the top.
- For **older phones**, choose the third version from the top.

### 2. Download the Config You Need
- In this repository, select the `.json` file you need.
- Click on the file, then click **Download**.

### 3. Import into ByeByeDPI
1. Open the ByeByeDPI app.
2. Tap the gear icon (⚙️) in the top right corner.
3. Tap the three dots (⋮) → select **"Import settings"**.
4. Choose the downloaded `.json` file.

### 4. Activate the Bypass
- Go back to the main screen.
- Tap **"Start"**.

**Done!** Discord, YouTube, games, and other sites should now work without blocks.
